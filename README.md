# SVM-Machine-Learning
A simple machine learning project that classifies iris flowers into three species using a Support Vector Machine (SVM).

# Iris Classification using SVM

## 📌 Overview

This project uses a Support Vector Machine (SVM) model to classify iris flowers into three species:

* Iris Setosa
* Iris Versicolour
* Iris Virginica

## ⚙️ Tech Stack

* Python
* NumPy
* scikit-learn

## 📊 Workflow

1. Load dataset
2. Split into train/test
3. Train SVM model
4. Evaluate accuracy

## 🎯 Result

Achieved 100% accuracy on test data.

## 🚀 Learning Outcome

* Understood train-test split
* Learned SVM basics
* Practiced prediction evaluation
